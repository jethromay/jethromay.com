About

Hey! My name is Jethro May and I currently work full time as a full-stack PHP developer in sunny South Africa. I am passionate about building the web, open-source projects and sharing my knowledge with others.

My coding journey began in the early stages of 2013, when I made the switch from the music industry and decided to focus on programming.

I have been building websites and applications for the past 7 years, where I have been exposed to a variety of different languages and frameworks. I am extremely passionate about PHP and more specifically the Laravel and WordPress ecosystems where I spend a lot of my time.

When I am not coding I enjoy spending time with my friends and family, keeping fit and travelling.

Setup

  • Macbook Air (2020)

    This is my main development machine which I use for the majority of my work. Its surprisingly fast and extremely portable.

  • Visual Studio Code

    I used PHPStorm for many years but after purchasing Caleb Porzio's course, Make VS Code Awesome I made the switch and haven't looked back.

  • Valet

    Laravel Valet provides me with a minimal and extremely fast environment for my projects.

  • Docker

    I currently use Docker for some of my WordPress projects, I tend to use the same structure for each project, so this allows me to quickly spin up and reuse the same environment.

  • Tinkerwell

    I predominantly use this app when I need to interact with my databases, this tends to service most of my needs but I still find some use in TablePlus.

  • HELO

    I recently started using HELO for email testing and debugging, it makes this usually painful process across a variety of environments extremely easy.

  • TablePlus

    Sequel Pro was my go-to database management tool for some time but when I tried out TablePlus I fell in love. It is extremely easy to use, visually appealing and the features are fantastic.

  • iTerm2

    Nothing comes close to iTerm2, it is a personal favourite and one of the first things that get installed when setting up a new machine.

  • Alfred 4 + Powerpack

    I only recently started using Alfred and it has been a game-changer, I cannot imagine working without it.

  • GitHub

    Over the years I have used GitLab, Bitbucket and GitHub for various projects. These days all of my public and private repositories sit on GitHub.
